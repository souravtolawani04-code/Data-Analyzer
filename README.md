# Data Analyzer
A simple Python command-line program that takes a list of numbers from the user and calculates basic statistics: total, average, maximum, and minimum.

## Features
- Accepts a custom count of numbers from the user
- Calculates:
  - **Total** (sum of all numbers)
  - **Average** (mean of all numbers)
  - **Maximum** value
  - **Minimum** value

## Requirements
- Python 3.x

## How to Run
1. Clone or download this repository.
2. Open a terminal in the project folder.
3. Run the script:

   ```bash
   python data_analyzer.py
   ```

4. Follow the prompts:

   ```
   Data Analyzer
   -------------
   How many numbers you want to enter: 3
   Enter number: 10
   Enter number: 20
   Enter number: 30

   Total: 60
   Average: 20.0
   Maximum: 30
   Minimum: 10
   Thank You❤️
   ```

## Notes
- Input numbers must be integers.
- The number of entries (`count`) must be at least 1.
